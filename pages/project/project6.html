Space Station Escape

COMPLETED ON: 27.JULY.2021

Skills Involved: Unity C# Game Development Game Design 2D Platformer

Mentor: Brian Winn

Links:

The Unity Engine was used to create this 2D platformer game. This game has four levels and features such as different types of enemies and a dialogue system. A comprehensive in-game menu is also designed.

This game is about an astronaut who is trapped in a space station that is being attacked by aliens. The goal of the game is to advance to the next level of the station before time runs out. This is a simple game in which you must collect points and keys in order to escape.
